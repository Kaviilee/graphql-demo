# graphql-demo

> 这是一个使用 Express MongoDB Graphql 的 demo。

## required

`.env file`

| key | description |
| -- | -- |
| **DB_HOST** | host of db |
| **MONGO_INITDB_ROOT_USERNAME** | your db username |
| **MONGO_INITDB_ROOT_PASSWORD** | your db password |
| **SECRET_TOKEN** | for jwt |
